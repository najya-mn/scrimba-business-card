# Digital Business Card

A sleek, interactive digital business card built with HTML, CSS, and JavaScript. Clicking the card copies the email to the clipboard and displays a custom popup notification.



## Technologies Used

* HTML – Structure of the digital business card
* CSS – Styling, hover effects, and animations
* JavaScript – Interactivity: click-to-copy email \& popup notification

## Features

* Clickable Card: Click the card to copy email to clipboard
* Custom Popup Notification: A smooth popup confirms the email is copied
* Hover \& Active States: Subtle effects that make the card feel like a button
* Responsive Design: Looks good on desktop and mobile
